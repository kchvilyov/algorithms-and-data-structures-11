## Практическое задание 11
К заданию приложен код некоторого метода и тесты для этого метода. Постарайтесь выполнить рефакторинг этого кода так,
чтобы он соответствовал концепции чистого кода.

## Критерий приема
Для работы требуется Java 17.
Запуск тестов выполняется через `./gradlew test`. Задание считается выполненным, если все тесты успешно пройдены.
